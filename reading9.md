# Reading 9

The command line is like the hub for telling your computer what to do without needing to use gui navigation or a mouse. It is super important to know the different prompts and commands that can be used here, and to become familiar with directing the computer or navigating and manipulating files through this window.

1. What is the Command Prompt?

The command prompt is the application in windows OS used to enter commands for the computer, usually  for troubleshooting or administrative functions and adjustments.

2. How do I access the Command Prompt?

To access the command prompt, use the shortcut in the start menu or on the apps screen, or search in the windows bar.

3. What are some of the most commonly used commands?

As per our reading, some of the most commonly used commands are chkdsk, copy, ftp, del, format, ping, attrib, net, dir, help and shutdown (lifewire.com/command-prompt-2625840)

4. What is Windows Power Shell?

The Windows Power Shell is a specific tool for automation and configuration management. (https://softwarekeep.com/help-center/powershell-terminal-command-prompt-windows)

5. What is Windows Terminal?

The Windows terminal is an interface for accessing different command-line tools (https://softwarekeep.com/help-center/powershell-terminal-command-prompt-windows)
